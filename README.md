# 项目简介
使用Vu3和Djnago搭建的校园网站，主要包括表白墙，二手交易，失误招领，在线聊天室，智慧班级,PC端和移动端分别使用element-plus和·vant组件库

## 主要功能
- 用户注册登录退出，个人信息修改，找回密码
- 实时在线聊天
- 表白墙动态发布，评论点赞，收藏
- 二手交易，交易信息发布，删除，修改
- 智慧班级，班级成员管理，作业收缴，投票
- 转盘，骰子，你画我猜游戏

## 技术栈
- Vue3,Vue-Router,Vue-x,axios,Websocket
- element-plus,vant
- Python Django Mysql
- Djangorestframkwork
